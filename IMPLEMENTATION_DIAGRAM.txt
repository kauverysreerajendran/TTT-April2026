SINGLE MODEL FLOW (Existing - Unchanged)
─────────────────────────────────────────────────────────────────────────────────
  User Loads Jig Modal
         ↓
  InitJigLoad API (lot_id, batch_id, jig_capacity)
         ↓
  Backend: Fetches Primary Tray List (lot_id)
         ↓
  Backend: Calculates Delink (with broken hooks)
         ↓
  Response: delink_tray_info (backward compat)
         ↓
  UI: Renders Primary Model Trays Only


MULTI-MODEL FLOW (New - Addition)
─────────────────────────────────────────────────────────────────────────────────
  User Loads Primary Model
         ↓
  User Clicks "Add Model"
         ↓
  Selects Secondary Model from JigView
         ↓
  Stores Secondary Lot in sessionStorage
         ↓
  Redirect with merge_model=1 flag
         ↓
  InitJigLoad API (lot_id, batch_id, multi_model=true, secondary_lots=[...])
         ↓
  Backend: Primary Allocation
         │ Fetches trays WHERE lot_id = primary_lot_id
         │ Allocates up to primary_qty (50)
         │ Tracks used_tray_ids {JB-A00001-JB-A00005}
         ↓
  Backend: Secondary Allocation (per secondary model)
         │ Fetches trays WHERE lot_id = secondary_lot_id
         │ Skip any tray_id in used_tray_ids (deduplication!)
         │ Allocates up to secondary_qty (48)
         │ Adds to used_tray_ids {JB-A00006-JB-A00009}
         ↓
  Backend: Validation
         │ Check: no duplicate tray_ids ✓
         │ Check: total <= effective_jig_capacity ✓
         ↓
  Response: 
         │ multi_model_allocation: [
         │   {model: "1805NAR02", allocated_qty: 50, tray_info: [...]},
         │   {model: "1805QBK02/GUN", allocated_qty: 48, tray_info: [...]}
         │ ]
         │ delink_tray_info: [...] (backward compat)
         ↓
  UI: Renders Both Models' Trays (grouped per model)
